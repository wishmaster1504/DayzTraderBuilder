Входные параметры:
- Можно задать путь к конфиг файлу с указанием его имени.


Что надо:
Конфиг файл (по умолчанию используем имя TraderBuilder.config)
Конфиг файл с суммами md5 по трейд файлам (автогенерируемый с режимом перезаписи MD5 после обновления файлов)
Логирование в файл надо?

Ограничения:
- создать папку, в которую положить (можно в папке с трейдером):
   1. Файл запуска программы
   2. Создать папку "/OUT"
   3. В папку "/PathTraderFiles" положить конфиг файлы всех трейдеров (сюда же будем сохранять файл с MD5)
   4. Готовый файл трейдера будет сформирован в папку "/OUT"
   5. В именах файлов трейдера НЕ использовать пробелы
    
Содержание конфиг файла
- путь к файлам трейдера
- список файлов трейдера (задает порядок добавления их в основной файл)
- основное имя файла трейдера (TraderConfig.txt) 
- полный путь куда копировать сгенерированный (продумать возможность задать больше 1-го пути. Например для двух одинаковых серверов)

Содержание Конфиг файл с суммами md5 по трейд файлам
- автогенерируемые записи суммы MD5 по каждому файлу трейдера


Алгоритм

Определяем текущий каталог, откуда вызвана программа.
Читаем конфиг файл
Проверяем изменялись ли файлы трейдера и не добавились ли новые (MD5)
Если файлы не менялись, то завершаем работу (новый файл не генерим)

Удаляем существующий файл трейдера в папке /IN (если есть)
Создаем файл трейдера в папке /IN
Пишем начальный тег трейдера
Проходим по каждому элементу массива файлов
Читаем файл и тупо копируем все содержимое (в режиме добавления)
Пишем заключительный тег файла трейдера (EndFile)
Закрываем файл трейдера

Копируем файл трейдера в папку мода с заменой

Перезаписать новые суммы MD5 по файлам в конфиг MD5

Конец программы
